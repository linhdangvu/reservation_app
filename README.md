## Installation environement
1. Install EXPO
```bash
npm install expo-cli --global
```
2. Init project
```bash
expo init project-test
```
3. Install webpack
```bash
expo install @expo/webpack-config@^0.17.0
```
4. Run project
```bash
npm start 
```
ou
```bash
yarn start
```
